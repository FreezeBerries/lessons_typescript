### notes from video:

how to install typescript:

```bash
npm i -g typescript
```

check for version:

```bash
tsc --version
```

how to compile typescript to javascript:

```bash
tsc index.ts
```

Not all libraries have typescript with them, but the community may maintain the types in another module:

```bash
npm i lodash
npm i -D @types/lodash
```
